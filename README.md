# Sisyphus The Czar

Стадия: POC B

Интерактивная веб-миниатюра: пользователь запускает падение камня, затем поднимает его из нижней тёмной сцены к верхней цели.

## Локальный запуск

Откройте `index.html` в браузере или запустите статический сервер:

```bash
python3 -m http.server 8080
```

После запуска через Python страница будет доступна на `http://127.0.0.1:8080/`.

## Docker

```bash
docker compose up -d --build
```

Контейнер слушает только локальный интерфейс сервера: `127.0.0.1:18082`.

## Сценарий

Страница стартует в светлой теме. Первый клик по камню включает физику: камень падает вниз, а страница скроллится к нижней зоне. После падения включается тёмная тема.

Панель параметров открывается кнопкой в правом верхнем углу. В ней регулируются масса камня, тяготение и сила руки. От этих значений зависит время удержания и высота подъёма. Победа фиксируется, когда камень помещён в верхнюю цель.
