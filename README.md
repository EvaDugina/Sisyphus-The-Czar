# Sisyphus The Czar

Стадия: POC B

Интерактивная веб-миниатюра: пользователь перетаскивает камень Сизифа, а страница имитирует его падение.

## Локальный запуск

Откройте `index.html` в браузере или запустите статический сервер:

```bash
python3 -m http.server 8080
```

После запуска через Python страница будет доступна на `http://127.0.0.1:8080/`.

## Docker

```bash
docker compose up -d --build
```

Контейнер слушает только локальный интерфейс сервера: `127.0.0.1:18082`.

## Деплой

На сервере проект разворачивается в `/home/zamay86/sisyphus-the-czar` и публикуется через nginx по адресу:

```text
http://82.202.157.20/sisyphus-the-czar/
```

nginx проксирует `/sisyphus-the-czar/` на `http://127.0.0.1:18082/`. Существующие маршруты `/monday/` и `/dashboard/` не меняются.
