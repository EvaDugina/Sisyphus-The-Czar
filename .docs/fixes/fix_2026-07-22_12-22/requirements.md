## Исходный текст

Пользователь просит:

- Ослабить вертикальную инерцию: даже при `inertia = 0.1` инерция вверх слишком сильная.
- Исключить проскальзывание по низу при `groundFriction = 1` при любой инерции.
- Заменить настройку "Сбрасывать линию при новом броске" на "Сбрасывать при касании земли".

---

## Функциональные требования

- **Ослабление вертикальной инерции:** Система должна уменьшить силу инерционного броска вверх так, чтобы при значении инерции `0.1` вертикальный импульс не воспринимался как чрезмерно сильный.
  - **Пояснение:** Минимальная инерция должна давать мягкое, контролируемое движение вверх.
  - **Источник:** "Инерция вверх даже при 0.1 слишком сильная."
  - **Acceptance hint:** При `inertia = 0.1` отпускание камня вверх дает заметно меньший вертикальный импульс, чем сейчас.

- **Полное отсутствие проскальзывания при трении 1:** Система должна исключать горизонтальное проскальзывание камня по низу сцены при `groundFriction = 1` при любом значении инерции.
  - **Пояснение:** Максимальное трение должно полностью гасить скольжение по земле.
  - **Источник:** "При трении 1 проскальзывания быть не должно при любой инерции."
  - **Acceptance hint:** При `groundFriction = 1` и любом `inertia` после отпускания/касания земли горизонтальная скорость на низу не приводит к сдвигу камня.

- **Сброс линии при касании земли:** Система должна заменить поведение настройки сброса линии: вместо сброса при новом броске линия должна сбрасываться при касании камнем земли.
  - **Пояснение:** След должен очищаться в момент контакта с низом сцены, а не в момент начала нового броска.
  - **Источник:** "Сбрасывать линию при новом броске измени на 'Сбрасывать при касании земли'"
  - **Acceptance hint:** Включенная настройка очищает линию при достижении земли; начало нового броска само по себе линию не сбрасывает.

- **Переименование настройки сброса:** Система должна изменить текст настройки с "Сбрасывать линию при новом броске" на "Сбрасывать при касании земли".
  - **Источник:** "измени на 'Сбрасывать при касании земли'"
  - **Acceptance hint:** В UI отображается новый текст настройки.

## Нефункциональные требования

- **Предсказуемость физики:** Значения `inertia` и `groundFriction` должны вести себя ожидаемо на крайних значениях шкалы.
  - **Пояснение:** `groundFriction = 1` должно быть стабильным режимом полного отсутствия скольжения по земле.
  - **Источник:** "При трении 1 проскальзывания быть не должно при любой инерции."
  - **Acceptance hint:** `groundFriction = 1` стабильно означает отсутствие скольжения по земле.

- **Сохранение управляемости:** Ослабление вертикальной инерции не должно полностью ломать возможность бросать камень вверх при средних и высоких значениях инерции.
  - **Пояснение:** Правка должна уменьшить чрезмерность минимальной инерции, но не обнулить механику броска.
  - **Источник:** "Инерция вверх даже при 0.1 слишком сильная."
  - **Acceptance hint:** При больших значениях `inertia` вертикальный бросок остается заметным.

## Проблемы и замечания, превращённые в требования

- **Слишком сильный вертикальный импульс:** Сейчас даже `inertia = 0.1` дает чрезмерную инерцию вверх. Требуемое поведение: минимальная инерция должна заметно ослаблять вертикальный импульс.
  - **Источник:** "Инерция вверх даже при 0.1 слишком сильная."
  - **Acceptance hint:** Значение `inertia = 0.1` больше не создает чрезмерный бросок вверх.

- **Максимальное трение не останавливает скольжение:** Сейчас при `inertia = 0.1` и `groundFriction = 1` все равно происходит проскальзывание по низу. Требуемое поведение: при `groundFriction = 1` скольжение по низу отсутствует при любой инерции.
  - **Источник:** "Даже при инерции 0.1 и трении 1 происходит проскальзывание по низу."
  - **Acceptance hint:** При `groundFriction = 1` горизонтальная скорость на земле гасится без видимого проскальзывания.

- **Сброс следа привязан не к тому событию:** Сейчас настройка сбрасывает линию при новом броске. Требуемое поведение: сбрасывать линию при касании земли.
  - **Источник:** "Сбрасывать линию при новом броске измени на 'Сбрасывать при касании земли'"
  - **Acceptance hint:** Включенная настройка очищает след при контакте камня с землей.
